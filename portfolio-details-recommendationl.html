Recommendation System on Steam

  • Category: Class Project
  • Class: Deep Learning
  • Date: Spring 2025

Description

Based on the Steam Review Dataset, we built a two phase recommendation system, beginning with a Recall stage and applies both collaborative filtering and a dual tower model to produce a shortlist of the top 200 candidate items. In the subsequent Ranking stage, these candidates are rescored using another dual tower network in combination with a multi gate mixture of experts (MMoE) model.
